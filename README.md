# Pok-mon-Explorer-


A simple Pokémon search app using [PokéAPI](https://pokeapi.co/).

## Features
- Search Pokémon by name
- Display image, ID, type, abilities, height, and weight
- Responsive design

## Setup
1. Clone the repo.
2. Open `index.html` in your browser.
3. Or deploy to GitHub Pages.

## Deployment
- Push this repo to GitHub.
- Go to **Settings → Pages → Source → main branch → /root**.
- Your app will be live at `https://your-username.github.io/pokemon-explorer`.
